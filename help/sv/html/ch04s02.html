Parametriska grafer

I ”Skapa graf”-fönstret kan du också välja flikhäftet Parametrisk för att skapa tvådimensionella parametriska grafer. På detta sätt kan du rita en parametrisk funktion. Du kan antingen ange punkterna som x och y, eller ange ett ensamt komplext tal som en funktion av variabeln t. Intervallet för variabeln t anges explicit, och funktionen ritas enligt angivet inkrement. Intervallet för x och y kan ställas in automatiskt genom att kryssa i kryssrutan Passa till beroende axel, eller så kan det anges explicit. Se Figure 4.3, “Flik för parametriska grafer”.
