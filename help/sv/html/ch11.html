To get help on a specific function from the console type: 

help FunctionName

Kommandon

help
help
help Funktionsnamn

Skriv ut hjälp (eller hjälp om funktion/kommando).

load
load "fil.gel"

Läs in en fil i tolken. Filen kommer att exekveras som om den skrevs in i kommandoraden.

cd
cd /katalog/namn

Ändra arbetskatalog till /katalog/namn.

pwd
pwd

Skriv ut aktuell arbetskatalog.

ls
ls

Lista filer i aktuell katalog.

plugin
plugin insticksmodulnamn

Läs in en insticksmodul. En insticksmodul med det namnet måste vara installerad på systemet i rätt katalog.
