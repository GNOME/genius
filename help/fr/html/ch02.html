Lancement de l'Outil de maths Genius

You can start Genius Mathematics Tool in the following ways:

Menu Applications

En fonction de votre système d'exploitation et de sa version, l'élément de menu pour l'Outil de maths Genius peut apparaître à différents endroits. Cela peut être dans les sous-menus Éducation, Accessoires, Bureautique, Science ou d'autres similaires en fonction de vos réglages personnels. Le nom de l'entrée de menu que vous recherchez est Outil de maths Genius. Une fois localisé, cliquez dessus pour démarrer l'Outil de maths Genius.

Boîte de dialogue Lancer une application

Il se peut que l'entrée de menu ne soit pas accessible pour votre système d'exploitation. Dans ce cas, vous pouvez ouvrir la boîte de dialogue Lancer une application et saisir gnome-genius.

Ligne de commande

Pour démarrer la version GNOME de l'Outil de maths Genius, saisissez gnome-genius.

Pour démarrer seulement la version en ligne de commande, exécutez la commande genius. Cette version ne comprend pas d'environnement graphique et certaines fonctionnalités comme le tracé de graphiques ne sont pas disponibles.
