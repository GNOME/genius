Procédure de démarrage de GEL

Dans un premier temps, le programme recherche dans le répertoire d'installation les fichiers de la bibliothèque installés (la version compilée lib.cgel), puis il recherche dans le répertoire actuel, puis il essaye de charger un fichier non compilé appelé ~/.geniusinit.

If you ever change the library in its installed place, you’ll have to first compile it with genius --compile loader.gel > lib.cgel
