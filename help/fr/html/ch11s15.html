Polynômes

AddPoly
AddPoly (p1,p2)

Ajoute deux polynômes (vecteurs).

DividePoly
DividePoly (p,q,&r)

Divise deux polynômes (en tant que vecteurs) en utilisant la division longue. Renvoie le quotient des deux polynômes. L'argument optionnel r est utilisé pour renvoyer le reste. Le reste a un degré plus bas que q.

See Planetmath for more information.

IsPoly
IsPoly (p)

Vérifie qu'un vecteur est utilisable en tant que polynôme.

MultiplyPoly
MultiplyPoly (p1,p2)

Multiplie deux polynômes (comme vecteurs).

NewtonsMethodPoly
NewtonsMethodPoly (poly,valeurinitiale,epsilon,maxn)

Find a root of a polynomial using Newton's method. poly is the polynomial as a vector and guess is the initial guess. The function returns after two successive values are within epsilon of each other, or after maxn tries, in which case the function returns null indicating failure.

See also NewtonsMethod.

Example to find the square root of 10: 

genius> NewtonsMethodPoly([-10,0,1],3,10^-10,100)

See Wikipedia for more information.

Poly2ndDerivative
Poly2ndDerivative (p)

Renvoie la dérivée seconde du polynôme (comme vecteur).

PolyDerivative
PolyDerivative (p)

Prend la dérivée du polynôme (comme vecteur).

PolyToFunction
PolyToFunction (p)

Fabrique une fonction à partir d'un polynôme (comme vecteur).

PolyToString
PolyToString (p,var...)

Fabrique une chaîne à partir d'un polynôme (comme vecteur).

SubtractPoly
SubtractPoly (p1,p2)

Soustrait deux polynômes (comme vecteur).

TrimPoly
TrimPoly (p)

Tronque les zéros d'un polynôme (défini comme vecteur).
