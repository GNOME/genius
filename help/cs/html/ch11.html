To get help on a specific function from the console type: 

help FunctionName

Příkazy

help
help
help NázevFunkce

Vypsat nápovědu (nebo nápovědu k funkci/příkazu).

load
load "soubor.gel"

Načíst soubor do interpretru. Obsah souboru bude proveden stejně, jako by byl napsán do příkazového řádku.

cd
cd /název/složky

Změnit pracovní složku na /název/složky.

pwd
pwd

Vypsat název aktuální pracovní složky.

ls
ls

Vypsat soubory v aktuální složce.

plugin
plugin název_zásuvného_modulu

Načíst zásuvný modul. Zásuvný modul se zadaným názvem musí být v systému nainstalován ve správné složce.
