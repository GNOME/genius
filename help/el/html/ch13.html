Για να ρυθμίσετε το Εργαλείο μαθηματικών Genius, επιλέξτε Ρυθμίσεις → Προτιμήσεις. Υπάρχουν αρκετές βασικές παράμετροι που παρέχονται από την αριθμομηχανή πέρα από αυτές που δίνονται από την τυπική βιβλιοθήκη. Αυτές ελέγχουν τη συμπεριφορά της αριθμομηχανής.

Αλλαγή ρυθμίσεων με την GEL

Πολλές από τις ρυθμίσεις στη Genius είναι απλά καθολικές μεταβλητές και μπορούν να υπολογιστούν και να εκχωρηθούν με τον ίδιο τρόπο όπως οι κανονικές μεταβλητές. Δείτε «Χρήση μεταβλητών» για τον υπολογισμό και την εκχώρηση σε μεταβλητές και «Παράμετροι» για έναν κατάλογο ρυθμίσεων που μπορούν να τροποποιηθούν με αυτόν τον τρόπο.

Ως παράδειγμα, μπορείτε να ορίσετε τον μέγιστο αριθμό ψηφίων σε ένα αποτέλεσμα σε 12 πληκτρολογώντας: 

MaxDigits = 12

Μέγιστος αριθμός ψηφίων στην έξοδο

Ο μέγιστος αριθμός ψηφίων σε ένα αποτέλεσμα (MaxDigits)

Αποτελέσματα ως αριθμοί κινητής υποδιαστολής

Αν τα αποτελέσματα πρέπει να εμφανίζονται πάντα ως αριθμοί κινητής υποδιαστολής (ResultsAsFloats)

Αριθμοί κινητής υποδιαστολής σε επιστημονική σημειογραφία

Αν οι αριθμοί κινητής υποδιαστολής πρέπει να είναι σε επιστημονική σημειογραφία (ScientificNotation)

Να εμφανίζονται πάντα πλήρεις παραστάσεις

Αν πρέπει να εμφανίζονται πλήρεις παραστάσεις για μη αριθμητικές τιμές επιστροφής (μεγαλύτερες από μια γραμμή) (FullExpressions)

Χρήση μικτών κλασμάτων

Αν τα κλάσματα πρέπει να εμφανίζονται ως μικτά κλάσματα όπως "1 1/3" αντί για "4/3". (MixedFractions)

Να εμφανίζεται 0.0 όταν ο αριθμός κινητής υποδιαστολής είναι μικρότερος από 10^-x (το 0=να μην περικόπτεται ποτέ)

How to chop output. But only when other numbers nearby are large. See the documentation of the parameter OutputChopExponent.

Να περικόπτονται οι αριθμοί μόνο όταν ένας άλλος αριθμός είναι μεγαλύτερος από 10^-x

When to chop output. This is set by the parameter OutputChopWhenExponent. See the documentation of the parameter OutputChopExponent.

Απομνημόνευση ρυθμίσεων εξόδου σε όλες τις συνεδρίες

Αν θα πρέπει οι ρυθμίσεις εξόδου στο Επιλογές εξόδου αριθμού/παράστασης να απομνημονεύονται για την επόμενη συνεδρία. Δεν εφαρμόζεται στο πλαίσιο Επιλογές εξόδου σφάλματος/πληροφοριών.

Αν δεν σημειωθεί, είτε η προεπιλογή είτε οποιεσδήποτε προηγουμένως αποθηκευμένες ρυθμίσεις χρησιμοποιούνται κάθε φορά που η Genius ξεκινά. Σημειώστε ότι οι ρυθμίσεις αποθηκεύονται στο τέλος της συνεδρίας, έτσι αν θέλετε να αλλάξετε τις προεπιλογές, σημειώστε αυτό το πλαίσιο, επανεκκινήστε Εργαλείο μαθηματικών Genius και έπειτα αποεπιλέξτε το ξανά.

Να εμφανίζονται τα σφάλματα σε διάλογο

Αν οριστεί τα σφάλματα θα εμφανίζονται σε έναν ξεχωριστό διάλογο, αν δεν οριστεί τα σφάλματα θα τυπωθούν στην κονσόλα.

Να εμφανίζονται τα μηνύματα πληροφοριών σε έναν διάλογο

Αν οριστεί τα μηνύματα πληροφοριών θα εμφανίζονται σε έναν ξεχωριστό διάλογο, αν δεν οριστεί τα μηνύματα πληροφοριών θα τυπωθούν στην κονσόλα.

Ο μέγιστος αριθμός σφαλμάτων που θα εμφανίζονται

Ο μέγιστος αριθμός σφαλμάτων που θα επιστρέφεται με έναν υπολογισμό (MaxErrors). Αν το ορίσετε σε 0, τότε όλα τα σφάλματα επιστρέφονται πάντα. Συνήθως, αν κάποιος βρόχος προκαλεί πολλά σφάλματα, τότε είναι απίθανο ότι θα μπορέσετε να πάρετε νόημα από περισσότερους εκτός από λίγους από αυτούς, έτσι βλέποντας έναν μεγάλο κατάλογο σφαλμάτων δεν είναι συνήθως χρήσιμο.

Πέρα από αυτές τις προτιμήσεις, υπάρχουν κάποιες προτιμήσεις που μπορούν να αλλαχθούν μόνο ορίζοντας τες στην κονσόλα χώρου εργασίας. Για άλλες που μπορούν να επηρεάσουν την έξοδο δείτε «Παράμετροι».

IntegerOutputBase

Η βάση που θα χρησιμοποιηθεί για την έξοδο ακεραίων

OutputStyle

A string, can be "normal", "latex", "mathml" or "troff" and it will affect how matrices (and perhaps other stuff) is printed, useful for pasting into documents. Normal style is the default human readable printing style of Genius Mathematics Tool. The other styles are for typesetting in LaTeX, MathML (XML), or in Troff.
