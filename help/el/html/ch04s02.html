Παραμετρικές γραφικές παραστάσεις

Στο παράθυρο δημιουργίας γραφικών παραστάσεων, μπορείτε επίσης να επιλέξετε την καρτέλα σημειωματαρίου Παραμετρικά για να δημιουργήσετε παραμετρικές γραφικές παραστάσεις δύο διαστάσεων. Με αυτόν τον τρόπο μπορείτε να σχεδιάσετε μια συνάρτηση με μια μοναδική παράμετρο. Μπορείτε είτε να ορίσετε τα σημεία ως x και y, είτε να δώσετε έναν μοναδικό μιγαδικό αριθμό ως μια συνάρτηση της μεταβλητής t. Η περιοχή της μεταβλητής t δίνεται ρητά και η συνάρτηση παίρνει δείγματα σύμφωνα με τη δοσμένη αύξηση. Η περιοχή x και y μπορεί να οριστεί αυτόματα ενεργοποιώντας το πλαίσιο ελέγχου Προσαρμογή εξαρτημένου άξονα, ή μπορεί να οριστεί ρητά. Δείτε Σχήμα 4.3, «Καρτέλα παραμετρικών γραφικών παραστάσεων».

Σχήμα 4.3. Καρτέλα παραμετρικών γραφικών παραστάσεων


Ένα παράδειγμα των παραμετρικών γραφικών παραστάσεων δίνεται στο Σχήμα 4.4, «Παραμετρικές γραφικές παραστάσεις». Παρόμοιες πράξεις μπορούν να γίνουν σε τέτοια γραφήματα όπως μπορούν να γίνουν σε άλλες γραμμικές γραφικές παραστάσεις. Για σχεδίαση χρησιμοποιώντας τη γραμμή εντολών δείτε την τεκμηρίωση της συνάρτησης LinePlotParametric ή LinePlotCParametric.
