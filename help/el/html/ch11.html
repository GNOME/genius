To get help on a specific function from the console type: 

help FunctionName

Εντολές

help
βοήθεια
βοήθεια FunctionName

Εμφάνιση βοήθειας (ή βοήθεια για μια συνάρτηση/εντολή).

load
load "file.gel"

Load a file into the interpreter. The file will execute as if it were typed onto the command line.

cd
cd /directory/name

Αλλαγή καταλόγου εργασίας σε /directory/name.

pwd
pwd

Εμφάνιση του τρέχοντος καταλόγου εργασίας.

ls
ls

Λίστα αρχείων στον τρέχοντα κατάλογο.

plugin
plugin plugin_name

Φόρτωση ενός προσθέτου. Το πρόσθετο πρέπει να εγκατασταθεί στο σύστημα στο σωστό κατάλογο.
