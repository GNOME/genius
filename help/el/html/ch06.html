Εξαρτήσεις

Σύνταξη: 

if <expression1> then <expression2> [else <expression3>]

Αν else παραλείπεται, τότε αν η expression1 δίνει ψευδές ή 0, επιστρέφεται η NULL.

Παραδείγματα: 

if(a==5)then(a=a-1)
if b<a then b=a
if c>0 then c=c-1 else c=0
a = ( if b>0 then b else 1 )

Σημειώστε ότι = θα μεταφραστεί σε == αν χρησιμοποιηθεί μέσα στην παράσταση για if, έτσι το 

if a=5 then a=a-1

θα ερμηνευτεί ως: 

if a==5 then a:=a-1
