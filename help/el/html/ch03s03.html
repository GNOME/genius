Για το άνοιγμα και την εκτέλεση ενός προγράμματος

Για να ανοίξετε ένα αρχείο, επιλέξτε Αρχείο → Άνοιγμα. Μια νέα καρτέλα που περιέχει το αρχείο θα εμφανιστεί στην περιοχή εργασίας. Μπορείτε να την χρησιμοποιήσετε για να επεξεργαστείτε το αρχείο.

Για να εκτελέσετε ένα πρόγραμμα από ένα αρχείο, επιλέξτε Αρχείο → Φόρτωση και εκτέλεση.... Αυτό θα τρέξει το πρόγραμμα χωρίς να το ανοίξει σε ξεχωριστή καρτέλα. Αυτό είναι ισοδύναμο με την εντολή φόρτωση.

If you have made edits to a file you wish to throw away and want to reload to the version that's on disk, you can choose the File → Reload from Disk menuitem. This is useful for experimenting with a program and making temporary edits, to run a program, but that you do not intend to keep.
