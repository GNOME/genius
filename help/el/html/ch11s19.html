SymbolicDerivative
SymbolicDerivative (f)

Προσπαθεί για συμβολική παραγώγιση της συνάρτησης f, όπου f είναι μια συνάρτηση μιας μεταβλητής.

Παραδείγματα: 

genius> SymbolicDerivative(sin)
= (`(x)=cos(x))
genius> SymbolicDerivative(`(x)=7*x^2)
= (`(x)=(7*(2*x)))

See Wikipedia for more information.

SymbolicDerivativeTry
SymbolicDerivativeTry (f)

Προσπαθεί να παραγωγίσει συμβολικά τη συνάρτηση f, όπου f είναι μια συνάρτηση μιας μεταβλητής, επιστρέφει null αν είναι ανεπιτυχής, αλλά είναι σιωπηλό. (Δείτε SymbolicDerivative)

See Wikipedia for more information.

SymbolicNthDerivative
SymbolicNthDerivative (f,n)

Προσπαθεί να παραγωγίσει συμβολικά μια συνάρτηση n φορές. (Δείτε SymbolicDerivative)

See Wikipedia for more information.

SymbolicNthDerivativeTry
SymbolicNthDerivativeTry (f,n)

Προσπαθεί να παραγωγίσει συμβολικά μια συνάρτηση n φορές σιωπηρά και επιστρέφει null στην αποτυχία (Δείτε SymbolicNthDerivative)

See Wikipedia for more information.

SymbolicTaylorApproximationFunction
SymbolicTaylorApproximationFunction (f,x0,n)

Προσπαθεί να κατασκευάσει τη συνάρτηση προσέγγισης Τέιλορ γύρω από το x0 στον nστο βαθμό. (Δείτε SymbolicDerivative)
