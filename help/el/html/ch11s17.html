Αντιμεταθετική άλγεβρα

MacaulayBound
MacaulayBound (c,d)

Για μια συνάρτηση Χίλμπερτ που είναι c για βαθμό d, με δεδομένο το πέρας Μακόλεϊ για τη συνάρτηση Χίλμπερτ βαθμού d+1 (Ο τελεστής c^<d> από την απόδειξη Γκριν).

Version 1.0.15 onwards.

MacaulayLowerOperator
MacaulayLowerOperator (c,d)

Ο τελεστής c_<d> από την απόδειξη Γκριν του θεωρήματος Μακόλεϊ.

Version 1.0.15 onwards.

MacaulayRep
MacaulayRep (c,d)

Επιστρέφει την dστή αναπαράσταση Μακόλεϊ ενός θετικού ακεραίου c.

Version 1.0.15 onwards.
