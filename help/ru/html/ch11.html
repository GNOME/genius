To get help on a specific function from the console type: 

help FunctionName

Команды

help
help
help ИмяФункции

Показывает справку (или справку по функции/команде).

load
load "file.gel"

Load a file into the interpreter. The file will execute as if it were typed onto the command line.

cd
cd /каталог/имя

Меняет рабочий каталог на /каталог/имя.

pwd
pwd

Выводит текущий рабочий каталог.

ls
ls

Показывает список файлов в текущем каталоге.

plugin
plugin plugin_name

Load a plugin. Plugin of that name must be installed on the system in the proper directory.
