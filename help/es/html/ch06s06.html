Variables de parámetros

Como se dijo, existen variables especiales llamadas parámetros. Éstas existen en todos los ámbitos. Para declarar un parámetro llamado foo con valor inicial 1, escriba 

parameter foo = 1

A partir de ahí, foo es una variable estrictamente global. Si asigna un valor a foo dentro de cualquier función, modificará la variable en todos los contextos. Esto significa que las funciones no tienen una copia privada de los parámetros.

Cuando define un parámetro utilizando la función undefine, éste deja de ser un parámetro.

Algunos parámetros están integrados de manera predeterminada y modifican el comportamiento de genius.
