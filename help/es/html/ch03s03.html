Abrir y ejecutar un programa

Para abrir un archivo, elija Archivo → Abrir. En el área de trabajo aparecerá una pestaña nueva que contiene al archivo. Puede usarla para editarlo.

Para ejecutar un programa desde un archivo, elija Archivo → Cargar y ejecutar.... Esto ejecutará el programa sin abrir en una pestaña separada. Es equivalente al comando load.

Si ha hecho cambios en un archivo y quiere descartarlos y volver a la versión original del disco, puede elegir la opción de menú Archivo → Recargar desde el disco. Esto es útil para experimentar con un programa y hacer cambios temporales, ejecutar el programa, pero no hacer que dichos cambios sean permanentes.
