Miscelánea

ASCIIToString
ASCIIToString (vec)

Convierte un vector de valores ASCII en una cadena. Consulte StringToASCII para más información.

Example: 

genius> ASCIIToString([97,98,99])
= "abc"

Consulte la Wikipedia para obtener más información.

AlphabetToString
AlphabetToString (vec,alfabeto)

Convertir un vector de valores alfanuméricos en base 0 (posiciones alfanuméricas de la cadena) a una cadena. Un null vector da como resultado una cadena vacía. Consulte mas información en StringToAlphabet.

Examples: 

genius> AlphabetToString([1,2,3,0,0],"abcd")
= "bcdaa"
genius> AlphabetToString(null,"abcd")
= ""

StringToASCII
StringToASCII (cad)

Convertir una cadena a un vector (fila) de valores ASCII. Consulte también ASCIIToString.

Example: 

genius> StringToASCII("abc")
= [97, 98, 99]

Consulte la Wikipedia para obtener más información.

StringToAlphabet
StringToAlphabet (str,alfabeto)

Convierte una cadena a un (fila) vector de valores alfabeticos en base 0 (posiciones en la cadena alfabetica), siendo -1 para caracteres desconocidos. Una cadena vacía será el resultado de una constante null. Consulte para más información en AlphabetToString.

Examples: 

genius> StringToAlphabet("cca","abcd")
= [2, 2, 0]
genius> StringToAlphabet("ccag","abcd")
= [2, 2, 0, -1]
