Al iniciar Genius

Cuando inicia la versión de la herramienta matemática Genius para GNOME, se muestra la ventana que aparece en la Figure 2.1, “Ventana de la herramienta matemática Genius”.

Figure 2.1. Ventana de la herramienta matemática Genius


La ventana de la herramienta matemática Genius contiene los siguientes elementos:

Barra de menú.

Los menús de la barra de menús contienen todos los comandos que necesita para trabajar con archivos en la Herramienta matemática Genius. El menú Archivo contiene elementos para cargar y guardar elementos y crear programas nuevos. El comando Cargar y ejecutar... no abre una ventana nueva para el programa, sino que lo ejecuta directamente. Es equivalente al comando load.

El menú Calculadora controla el motor de la calculadora. Permite ejecutar el programa actualmente seleccionado o interrumpir el cálculo actual. También puede mirar la expresión completa del último resultado (útil si el último resultado fue demasiado grande para ajustar a la consola), y puede mirar una lista con los valores de todas las variables definidas por el usuario. También puede monitorizar las variables del usuario, que es especialmente útil cuando se está ejecutando un cálculo muy largo, o para depurar un cierto programa. Finalmente, la Calculadora permite trazar funciones usando un cuadro de diálogo amigable.

El menú Ejemplos es una lista de programas de ejemplo o demostraciones. Si abre el menú, cargará el ejemplo en un nuevo programa, que puede ejecutar, editar, modificar y guardar. Estos programas deberían estar bien documentados y generalmente muestran alguna característica de Herramienta matemática Genius o algunos conceptos matemáticos.

El menú Programas muestra una lista de programas abiertos y le permite cambiar a uno de ellos.

Los otros menús tienen funciones similares a las de otras aplicaciones.

Barra de herramientas.

La barra de herramientas contiene un subconjunto de comandos a los que puede acceder desde la barra de menús.

Área de trabajo

El área de trabajo es el método primario para interactuar con la aplicación.

Inicialmente, el área de trabajo sólo tenía la pestaña Consola, que es el principal medio de interacción con la calculadora. Aquí puede escribir expresiones y el resultado se devuelve inmediatamente después de pulsar la tecla «Intro».

Alternativamente, puede escribir programas más largos y éstos pueden aparecer en pestañas separadas. Los programas son un conjunto de comandos o funciones que se pueden ejecutar a la vez sin tener que ejecutarlos desde una terminal. Los programas se pueden guardar en archivos para su posterior recuperación.
