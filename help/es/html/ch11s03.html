Parámetros

ChopTolerance
ChopTolerance = número

Tolerancia de la función Chop

ContinuousNumberOfTries
ContinuousNumberOfTries = número

Iteraciones para tratar de obtener el límite de continuidad y sus límites.

ContinuousSFS
ContinuousSFS = número

Número de pasos sucesivos dentro de la tolerancia para el cálculo de la continuidad.

ContinuousTolerance
ContinuousTolerance = número

Tolerancia para la continuidad de las funciones y para el cálculo del límite.

DerivativeNumberOfTries
DerivativeNumberOfTries = número

Número de iteraciones para tratar de obtener el límite para la derivada.

DerivativeSFS
DerivativeSFS = número

Número de pasos sucesivos dentro de la tolerancia para el cálculo de la derivada.

DerivativeTolerance
DerivativeTolerance = número

Tolerancia para calcular las derivadas de las funciones.

ErrorFunctionTolerance
ErrorFunctionTolerance = número

Tolerancia de ErrorFunction.

FloatPrecision
FloatPrecision = número

Precisión en coma flotante.

FullExpressions
FullExpressions = booleano

Imprimir expresiones completas, incluso si son de más de una línea.

GaussDistributionTolerance
GaussDistributionTolerance = número

Tolerancia de la función GaussDistribution.

IntegerOutputBase
IntegerOutputBase = número

Base de salida de enteros.

IsPrimeMillerRabinReps
IsPrimeMillerRabinReps = número

Número de tests de Miller-Rabin adicionales que ejecutar sobre un número antes de declararlo primo en IsPrime.

LinePlotDrawLegends
LinePlotDrawLegends = true

Le dice a Genius cómo dibujar las leyendas para las funciones para dibujar líneas tales como LinePlot.

LinePlotDrawAxisLabels
LinePlotDrawAxisLabels = true

Le dice a Genius que dibuje las etiquetas de los ejes cuando se utilizan funciones para dibujar líneas como LinePlot.

Desde la versión 1.0.16 en adelante.

LinePlotVariableNames
LinePlotVariableNames = ["x","y","z","t"]

Le dice a Genius qué nombres de variable se usan como nombres predeterminados para funciones para dibujar líneas tales como LinePlot y compañía.

Desde la versión 1.0.10 en adelante.

LinePlotWindow
LinePlotWindow = [x1,x2,y1,y2]

Establece los límites para lasfunciones para dibujar líneas tales como LinePlot.

MaxDigits
MaxDigits = número

Máximo de cifras que mostrar.

MaxErrors
MaxErrors = número

Máximo de errores que mostrar.

MixedFractions
MixedFractions = booleano

Si es cierto, las fracciones mixtas se imprimen.

NumericalIntegralFunction
NumericalIntegralFunction = función

La función usada para la integración en NumericalIntegral.

NumericalIntegralSteps
NumericalIntegralSteps = número

Pasos que realizar en NumericalIntegral.

OutputChopExponent
OutputChopExponent = número

Cuando otro número que se imprime en el objeto (una matriz o un valor) es mayor que 10-OutputChopWhenExponent, y el número que se imprime es menor que 10-OutputChopExponent, entonces se mostrará 0.0 en lugar del número.

La salida nunca se corta si OutputChopExponent es cero. El número será un entero positivo.

Si quiere que la salida siempre se corte según OutputChopExponent, configure OutputChopWhenExponent, para un valor mayor o igual a OutputChopExponent.

OutputChopWhenExponent
OutputChopWhenExponent = número

Cuando recortar la salida. Consulte OutputChopExponent.

OutputStyle
OutputStyle = cadena

Estilo de salida, puede ser normal, latex, mathml o troff.

Esto afecta principalmente a cómo las matrices y fracciones se imprimen y es útil para pegar en los documentos. Por ejemplo, puede configurarlo en LaTeX usando: 

OutputStyle = "latex"
ResultsAsFloats
ResultsAsFloats = booleano

Convertir todos los resultados a flotantes antes de imprimir.

ScientificNotation
ScientificNotation = booleano

Usar notación científica.

SlopefieldTicks
SlopefieldTicks = [vertical,horizontal]

Establece el número de pasos verticales y horizontales en un diagrama de pendientes. (Consulte SlopefieldPlot).

Desde la versión 1.0.10 en adelante.

SumProductNumberOfTries
SumProductNumberOfTries = número

Cuántas iteraciones probar para InfiniteSum y InfiniteProduct.

SumProductSFS
SumProductSFS = número

La cantidad de pasos consecutivos para estar dentro de los límites para InfiniteSum y InfiniteProduct.

SumProductTolerance
SumProductTolerance = número

Tolerancia para InfiniteSum y InfiniteProduct.

SurfacePlotDrawLegends
SurfacePlotDrawLegends = true

Le dice a Genius cómo dibujar las leyendas para las funciones de dibujado de superficie tales como SurfacePlot.

Desde la versión 1.0.16 en adelante.

SurfacePlotVariableNames
SurfacePlotVariableNames = ["x","y","z"]

Indica a Genius los nombres de variables que se usan como nombres predeterminados para las funciones de superficie de dibujado que utiliza SurfacePlot. Compruebe que z no se refiere al eje dependiente (vertical), sino que a la variable compleja independiente z=x+iy.

Desde la versión 1.0.10 en adelante.

SurfacePlotWindow
SurfacePlotWindow = [x1,x2,y1,y2,z1,z2]

Establece los límites de la superficie de dibujado (consulte SurfacePlot).

VectorfieldNormalized
VectorfieldNormalized = true

Indica si la longitud de las flechas tendrá un tamaño normalizado en el trazado dentro del campo vectorial. Si es cierto, en el campo vectorial solo aparecerá la dirección y no la magnitud. (Consulte VectorfieldPlot).

VectorfieldTicks
VectorfieldTicks = [vertical,horizontal]

Establece el número de pasos verticales y horizontales en un gráfico de campo de vectores. (Observe VectorfieldPlot).

Desde la versión 1.0.10 en adelante.
