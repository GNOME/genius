Polinomios

AddPoly
AddPoly (p1,p2)

Suma dos polinomios (vectores).

DividePoly
DividePoly (p,q,&r)

Dividir dos polinomios (como vectores) utilizando la división larga. Devuelve el cociente de los dos polinomios. El argumento opcional r se utiliza para devolver el residuo. El residuo tendrá el grado más bajo que q.

Consulte Planetmath para obtener más información.

IsPoly
IsPoly (p)

Comprobar si un vector se puede usar como un polinomio.

MultiplyPoly
MultiplyPoly (p1,p2)

Multiplica dos polinomios (como vectores).

NewtonsMethodPoly
NewtonsMethodPoly (poly,guess,epsilon,maxn)

Encontrar una raíz de un polinomio utilizando el método de Newton. La variable poly es el polinomio en forma vectorial y guess es la suposición inicial. La función devuelve después dos valores sucesivos que están dentro de los límites que marca epsilon o después de maxn iteraciones en cuyo caso devuelve null indicando un fallo.

Consulte también NewtonsMethod.

Ejemplo para encontrar la raíz cuadrada de 10: 

genius> NewtonsMethodPoly([-10,0,1],3,10^-10,100)

Consulte la Wikipedia para obtener más información.
