Genius Mathematikwerkzeug starten

Sie können Genius Mathematikwerkzeug auf folgende Arten starten:

Menü Anwendungen

Abhängig von Ihrem Betriebssystem und dessen Version kann sich der Menüeintrag für Genius Mathematikwerkzeug an verschiedenen Stellen befinden. Es kann sich unter Bildung, Zubehör, Büro, Wissenschaft oder einem ähnlichen Untermenü befinden, abhängig von den Einstellungen Ihres Systems. Der Menüeintrag, nach dem Sie suchen sollten, lautet Genius Mathematikwerkzeug. Sobald Sie diesen Menüeintrag gefunden haben, klicken Sie darauf, um Genius Mathematikwerkzeug zu starten.

Ausführen-Dialog

Es kann auch vorkommen, dass der Menüeintrag bei Ihrem System nicht vorhanden ist. In diesem Fall können Sie das Dialogfeld zum Ausführen von Programmen öffnen und den Befehl gnome-genius eingeben.

Befehlszeile

Um die GNOME-Version des Genius Mathematikwerkzeug von der Befehlszeile aus zu starten, geben Sie gnome-genius ein.

Zum Starten der Befehlszeilen-Version führen Sie folgenden Befehl aus: genius. Diese Version enthält keine graphische Umgebung und einige Funktionen wie zum Beispiel die Darstellung stehen nicht zur Verfügung.
